# TresMenosDos.uno

[TresMenosDos.uno](https://TresMenosDos.uno) is a lonely site for practicing mathematical operations like multiplication, division, exponents, and square roots. It employs some learning techniques to enhance mathematical skills.

## Key Features

- Covers key mathematical operations:
  - Multiplication tables
  - Division
  - Exponents (powers)
  - Square roots
- Emoji-based, minimalist interface for intuitive use across languages
- ChatGPT was kind enought to provide translations for some languages (English, Spanish, Portuguese, French, Italian)
- Learning algorithms:
  - Spaced repetition 
  - Error-weighted question selection
  - Sequential and random learning modes
- Focus on independent learning and error correction
- Mobile-first, responsive design

## Learning Techniques

The site employs several research-backed learning methods:

- Sequential and random learning algorithm
- Repetition-based approach with corrections
- Spaced repetition focused on errors
- Revisit probability for challenging questions
- Error-weighted algorithm
- Sequential reset after multiple errors

## International Focus

The little text in the site is provided in multiple languages. The emoji-based interface further reduces language barriers (we hope).

## Usage

Simply visit [TresMenosDos.uno](https://TresMenosDos.uno) and start practicing! The intuitive interface will guide you through mathematical exercises tailored to your performance.
